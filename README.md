# webapp-training-1

## Getting started

Make sure you have [Docker](https://www.docker.com/products/docker-desktop) installed.

### Start the app

```bash
docker compose up
```

This will start:
- **API** (Node.js) — available at http://localhost:8081
- **MongoDB** — available on port 27017

### Stop the app

```bash
docker compose down
```
